## 3. 파일이 어디에 있나
```
튜브폴더\튜브폴더\
├─ tubefolder-extension\          ← 원본 작업본 (미리보기 서버가 이걸 서빙)
├─ tubefolder-extension.zip
├─ _preview_server.js  + .claude\launch.json   (검증 서버, port 8731)
├─ youtube-manager-extension\     ← 버려진 빈 CRA 스텁 (건드리지 말 것)
└─ 결과물\                        ← 전달·보관용 사본 (소스+과정+관련자료+이 파일)
```
> ⚠️ 코드 수정은 **원본 `tubefolder-extension\`** 에서 하고, 끝나면 `결과물\tubefolder-extension\`로 다시 복사 + zip 재생성.
