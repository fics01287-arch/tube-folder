# 튜브폴더 (TubeFolder)

유튜브 동영상·뮤직을 **내 컴퓨터 탐색기처럼 무제한 폴더로 정리**하는 크롬 확장 프로그램입니다.
빌드 도구·외부 라이브러리 없이 순수 HTML/CSS/JS로 작성되어, **폴더를 zip으로 압축하면 그대로 배포**됩니다.

---

## 설치 (데스크톱 크롬/엣지)

1. 크롬 주소창에 `chrome://extensions` 입력 (엣지는 `edge://extensions`)
2. 오른쪽 위 **개발자 모드** 켜기
3. **압축해제된 확장 프로그램을 로드** 클릭 → `tubefolder-extension` 폴더 선택
   - (남에게 zip으로 받았다면 먼저 압축을 푼 뒤 그 폴더를 선택)
4. 툴바의 튜브폴더 아이콘 클릭 → 관리 화면이 새 탭으로 열립니다.

## 사용법

- **새 폴더 / 동영상 추가** : 상단 툴바 버튼. 빈 곳 우클릭으로도 가능.
- **유튜브에서 바로 담기** : 유튜브 영상/뮤직 페이지(또는 영상 링크)에서 **우클릭 → “튜브폴더에 추가”**.
- **이동** : 항목을 드래그해서 폴더/트리/주소줄(브레드크럼)에 떨어뜨리기.
- **여러 개 선택** : 빈 영역에서 **마우스로 드래그(고무줄 선택)**, 또는 `Ctrl`/`Shift`+클릭.
- **복사/잘라내기/붙여넣기** : 우클릭 메뉴 또는 `Ctrl+C` / `Ctrl+X` / `Ctrl+V`. (잘라낸 항목은 흐리게 표시)
- **이름 바꾸기** : 선택 후 `F2` 또는 우클릭. (같은 폴더에 이름이 겹치면 자동으로 `(2)` 부여)
- **삭제 / 되돌리기** : `Delete` → 휴지통. 실수했으면 **`Ctrl+Z`(실행취소)**. 휴지통에서 **복원 / 완전 삭제 / 휴지통 비우기**.
- **수동 순서** : 정렬을 ‘없음’으로 두고 `Alt+↑` / `Alt+↓` 로 항목 순서를 직접 조정.
- **여러 영상 한 번에 추가** : “동영상 추가”에 URL을 줄바꿈/공백으로 여러 개 붙여넣기. 브라우저에서 **유튜브 링크를 끌어다 놓아도** 추가됩니다.
- **정렬** : 이름 · 수정한 날짜 · 유형 · 크기 · 없음 × 오름/내림차순. (자세히 보기에선 열 머리글 클릭)
- **보기** : 아주 큰 아이콘 · 큰 아이콘 · 보통 · 작은 아이콘 · 목록 · 자세히.
- **검색** : 상단 검색창(전체 폴더에서 이름 검색).
- 정리 내용은 `chrome.storage.local`(+`unlimitedStorage`)에 저장되어 **확장을 껐다 켜도, 브라우저를 재시작해도 유지**됩니다.

### 단축키
`F2` 이름변경 · `Delete` 삭제 · `Ctrl+Z` 실행취소 · `Enter` 열기 · `Backspace` 상위폴더 · `↑↓←→` 선택이동 · `Alt+↑/↓` 수동순서 · `Ctrl+A` 전체선택 · `Ctrl+C/X/V` 복사/잘라/붙여넣기

## 휴대폰에서 쓰기 + 기기 연동

> ⚠️ **일반 모바일 크롬은 확장 프로그램을 지원하지 않습니다(브라우저 자체 한계).**

그래서 두 가지를 제공합니다.

1. **모바일 PWA(웹앱)로 설치** — 이 폴더는 **PWA로 만들어져** 있습니다(`manager.webmanifest`+`sw.js`).
   폴더를 정적 호스팅(예: GitHub Pages, Netlify, 또는 사내 서버)에 https로 올린 뒤 폰에서 그 주소를 열고 **“홈 화면에 추가”** 하면 앱처럼 전체화면으로 쓰이고 **오프라인에서도 동작**합니다. (저장은 그 브라우저의 localStorage)
2. **기기 간 동기화 = 내보내기/가져오기** — 데스크톱에서 **⬆ 내보내기**로 `tubefolder-backup.json` 저장 → 폰에서 **⬇ 가져오기**로 불러오면 정리 상태가 그대로 옮겨집니다. (반대 방향도 동일)

> **자동 실시간 동기화**(따로 안 옮겨도 양쪽이 자동 일치)는 백엔드가 필요합니다. 권장은 **구글 드라이브 `appDataFolder`**(본인 계정·무료·무서버). 자세한 비교는 [docs/GAP-ANALYSIS-AND-ROADMAP.md](docs/GAP-ANALYSIS-AND-ROADMAP.md#5-모바일자동-동기화-옵션).

## 남에게 전달 (단일 파일 배포)

`tubefolder-extension` 폴더를 통째로 zip으로 압축해 전달하면 됩니다. (`node_modules`·빌드 산출물 없음)
받는 사람은 압축을 풀고 위 **설치** 절차대로 “압축해제된 확장 로드”만 하면 됩니다.

---

## 폴더 구성

```
tubefolder-extension/
├─ manifest.json     확장 정의 (MV3)
├─ background.js     서비스워커: 아이콘 클릭, 우클릭 "튜브폴더에 추가"
├─ manager.html      관리 화면
├─ manager.css       스타일 (보기 모드/반응형/마퀴 포함)
├─ app.js            핵심 로직 (트리·정렬·드래그·휴지통·복사·검색·실행취소)
├─ storage.js        저장 계층 + 헬퍼 (확장=chrome.storage / 웹=localStorage)
├─ manager.webmanifest  PWA 설치 정의 (모바일 웹앱)
├─ sw.js             PWA 오프라인 캐시 (웹 전용, 확장과 무관)
├─ icons/            아이콘 16·48·128·192·512
├─ AGENTS.md         🤖 AI 에이전트 자가진화 지침 (오류수정·개선 절차)
└─ docs/             설계 문서 (아래)
```

## 설계 문서 (docs/)

- [AGENTS.md](AGENTS.md) — 🤖 **자율 에이전트용** 실행·검증 절차, 코드 지도, 불변식, 회귀 테스트, 함정
- [ARCHITECTURE.md](docs/ARCHITECTURE.md) — 전체 구조·런타임 컨텍스트·데이터 흐름·저장 전략·권한
- [DATA-MODEL.md](docs/DATA-MODEL.md) — 노드 스키마·불변식·특수 노드·버전/마이그레이션
- [ALGORITHMS.md](docs/ALGORITHMS.md) — 정렬/트리/이동·복사·삭제/마퀴/실행취소/메타수집 알고리즘 + 복잡도
- [GAP-ANALYSIS-AND-ROADMAP.md](docs/GAP-ANALYSIS-AND-ROADMAP.md) — 발견한 부족분·v1.1 보완 내역·남은 과제·모바일 동기화 옵션

## 동작 메모

- 유튜브 재생목록은 본래 평면 구조라, **중첩 폴더는 이 확장의 자체 저장소**에 보관합니다. 동영상 항목은 ID·URL·썸네일을 참조하며 더블클릭하면 유튜브에서 열립니다.
- “크기” 정렬: 동영상=재생시간, 폴더=하위 항목 수 기준입니다.
- 휴지통은 항상 최상위 폴더의 맨 아래에 고정됩니다.
- 메타데이터(제목)는 API 키 없이 oEmbed로 가져오며, 모바일 웹에서는 `noembed.com`으로 자동 폴백합니다.
